차원축소(Dimension Reduction)
고차원 데이터로부터, 저차원 의 데이터로 변환하는 방법.

왜 하는가? 
--> 특징이 많을 경우, 개별 특징간에 상관관계가 높을 가능성이 큼.
--> 다중 공선성 문제로 인해 모델의 예측 성능이 저하될 가능성이 크다.


차원축소의 종류
- PCA(Principal Component Analysis) : 주성분 분석
- LDA(Latent Dirichlet allocation) : 잠재 디리클레 할당
- SVD(Singular Value Decomposition) : 특이값 분해
- NMF(Non-negative matrix factorization) : 음수 미포함 행렬 분해

장점 
1. 일반화 성능 향상 (이미지일 경우)
└ 차원이 축소된 이미지는 원본 이미지보다 훨씬 작은 차원이므로
  이미지 분류 등의 분류 작업 수행 시 과적합(overfitting)의 영향력 작음.
2. 텍스트 문서의 숨겨진 의미 추출
└ 차원 축소 알고리즘을 통해 문서 내 단어들의 구성에
  숨겨진 Semantic 의미나 Topic 잠재요소를 간주하고 이를 찾아낼 수 있음.

PCA (주성분 분석)
- 고차원의 데이터를 저차원으로 변환할 수 있는 알고리즘
- 서로 연관성이 있는 고차원 변환을 직교 변환을 사용하여 저 차원 공간 (주성분)의표본으로 변환
- 데이터를 한개의 축으로 투영했을 때 , 분산이 가장 커지는 축을 첫번째 주성분
- 두번째로 커지는 축을 두번째 주성분에 놓이도록 새로운 좌표계로 데이터를 선형변환
- 주성분들은 직교한다는 제약아래에 가장 큰 분산을 갖고 있다
